# Certified Cyber Security Analyst (C3SA)

The C3SA certification introduced the major technical areas a cybersecurity analyst encounters, from understanding attacks to protecting systems and supporting security operations. Its broad scope helped me connect offensive techniques with the defensive decisions an analyst needs to make.

## What it covered

- Cybersecurity concepts, career paths, and rules of engagement
- Building a local cyber range with attacker and vulnerable systems
- Web fundamentals, common misconfigurations, OWASP Top 10 vulnerabilities, and web testing methodology
- Network protocols, architecture mapping, exploitation, pivoting, and network testing methodology
- Windows and Linux command-line fundamentals and host exploitation
- Cloud fundamentals across AWS and multi-cloud environments
- Host and network defence, threat intelligence, threat hunting, and incident response
- Security operations fundamentals and a final assessment

## What I gained

This certification improved my understanding of how web, network, host, and cloud weaknesses appear from both attacker and defender perspectives. It also strengthened the security-operations mindset I use when reviewing evidence, investigating suspicious activity, and considering practical mitigations.
