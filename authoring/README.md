# Authoring workspace

Nothing in this folder is published to the website.

- `templates/` contains copyable Markdown starting points.
- `drafts/` contains unfinished writing that is not ready for a collection.
- `source-images/` contains large originals used to create optimized public images.
- `archive/` contains old exports, duplicate files, and retired assets kept for reference.

Published writing belongs in the appropriate root collection such as `_quacks`, `_ctf`, `_projects`, or `_tools`.
